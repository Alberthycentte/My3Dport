Create a new GitHub repo and push these files. 2. In GitHub repo settings -> Pages -> choose branch `main` (or `gh-pages`) and `/ (root)`. 3. Visit `https://.github.io//` ## How to add/edit a project - Add files (models/step/gerber) into the `assets/` folder. - Edit `projects.json` to add a new project object (see example). - Commit changes to GitHub. Pages will auto-deploy. ## Notes about preventing downloads Completely preventing downloads is impossible: the browser needs the file. This starter hides direct links, disables right-click, and obfuscates paths. For stronger protection consider a server-side fetch-with-token or watermarking renders. ``` --- FILE: index.html ```html My 3D & PCB Design Portfolio
